# mesh — prebuilt CLI bundles (v1.1.0-6ba5e5a)

Self-contained `mesh` + `meshl` bundles. Run on any **Bun** or **Node 18+**; no native
binary, nothing to compile.

## Install / update

```sh
bun add -g github:shizlie/libTwoToOne        # or: npm i -g github:shizlie/libTwoToOne
```

This puts `mesh` and `meshl` on your PATH. Re-run to update — if it still looks stale, run
`bun pm cache rm` first (Bun caches global git installs).

## Run the live agent demo (optional)

This repo also ships `live-demo.sh` and the two agent operating contracts it loads. With a
logged-in `claude` plus `tmux` and `bun`, stand up two real Claude agents that coordinate
through a room:

```sh
git clone --depth 1 https://github.com/shizlie/libTwoToOne demo && cd demo
ROOM_URL=https://<your-room> bash live-demo.sh        # then: bash live-demo.sh --fire
```

Nothing is scripted — each agent is a live `claude` acting on its contract, woken by its own
`meshl` listener. Tear down with `bash live-demo.sh --clean`.
